# **App Name**: PredictPro

## Core Features:

- User Authentication: Secure user registration and login with OTP verification, role-based access control (User, Admin, Assistant, SuperAdmin), and device binding.
- Payment & License System: MPESA and Airtel Money payment integration with transaction message parsing, admin verification queue, license generation with game-specific access, booking fee system, and automated license expiration and renewal.
- Game Prediction Engine: AI-driven generation of game-specific predictions (Aviator, Crash, Mines, Gems) with real-time analysis, probability heatmaps, confidence scoring, and volatility assessment. Use of a tool that adapts prediction models based on user feedback.
- Subscription Management: Management of premium subscription tiers (Basic, Standard, Pro Plus) with plan-specific features, license activation, countdown timers, and automated expiration handling.
- Support System: Multi-channel support system (System Help, Manager, Customer Care) with an automated assistant, ticket management, and escalation workflows. Assistant invite system with single-use tokens.
- Admin Dashboard: Comprehensive admin interface for payment verification, user management, license control, analytics dashboards, system settings, and fraud detection.
- Referral & Rewards: User referral tracking with commission calculations, task-based reward system, wallet management, and automated reward distribution. Management of commission tracking for assistants.

## Style Guidelines:

- Primary color: Dark background (#0f0f0f) to provide a modern, premium aesthetic suited for a gaming app.
- Secondary color: White (#ffffff) used for primary text, ensuring readability against the dark background.
- Basic accent: Sky blue (#38bdf8) used to highlight interactive elements in the Basic tier, evoking trust and security.
- Standard accent: GitHub blue (#1F6FEB) to distinguish interactive elements and actions for the Standard tier, associating it with a sense of reliability.
- Pro Plus accent: Amber/Gold (#F59E0B) used for the highest tier to reflect premium value and sophistication.
- Headings: 'Inter', a grotesque-style sans-serif, known for its modern, machined, and objective feel, to create clear and impactful headlines. If longer text is anticipated, it should be paired with 'Inter' for body.
- Accent text: 'Source Code Pro' a monospaced sans-serif font, which works very well for displaying codes and other similar content.
- Lucide React for consistent, clear, and modern icons, enhancing the app's user interface with intuitive visual cues. Use premium badge indicators with metallic effects, adding sophistication to the interface.
- Responsive PWA layout optimized for a mobile-first experience, featuring smooth animations, glass-morphism effects, and tier-based color accents.
- Loading animations with branded messaging that enhance user engagement by providing feedback on system operations. Subtle transitions and animations used throughout the application to improve user experience.
